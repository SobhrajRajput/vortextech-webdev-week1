# Personal Portfolio Website

## About This Project
This project is a responsive personal portfolio website created as part of the Week 1 HTML and CSS task. It includes my introduction, skills, and contact information in a clean and simple design.

## Features
- Header section with my name and short title
- About section with a brief introduction
- Skills section using a list format
- Contact section with contact information
- Responsive design for desktop and mobile screens
- CSS styling with Flexbox layout

## Technologies Used
- HTML5
- CSS3
- CSS Flexbox
- Responsive Design

## How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in any web browser.
4. The portfolio website will load successfully.
